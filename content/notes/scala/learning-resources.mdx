Like any new technology, Scala and its ecosystem take a while to master and can be frustrating at times. Don't get discouraged! Here are some resources to help.

- Popular IDEs are IntelliJ IDEA with the Scala plugin and Visual Studio Code with the Scala (Metals) extension. If you're unsure which to choose, go with IntelliJ.
- [Tour of Scala](https://docs.scala-lang.org/tour/tour-of-scala.html) - A tour of bite-sized introductions to the most frequently used features of Scala intended for newcomers to the language.
- [Scala: The Good Parts](https://rocky.dev/scala-the-good-parts) - Scala allows you to accomplish tasks in different ways. This blog post explains some of the more readable, maintainable approaches.

### Tools/Libraries

- [sbt](https://www.scala-sbt.org/index.html) - Scala Build Tool, similar to Gradle for Java.
- [Scalafmt](https://scalameta.org/scalafmt/) - Code formatter, similar to Prettier for JavaScript.
- [Scalafix](https://scalacenter.github.io/scalafix/) - Refactoring and linting tool, similar to ESLint for JavaScript.
- [ScalaTest](https://www.scalatest.org/) - Testing tool, similar to JUnit for Java.
- [mockito-scala](https://github.com/mockito/mockito-scala) - Test mocking, a Scala-friendly API over Java's Mockito.
- [Cats](https://typelevel.org/cats/) - Short for "category", provides functional programming abstractions. The homepage provides Type Class and Date Type documentation along with motivations behind the library.
  - ðŸ“– [Scala with Cats](https://www.scalawithcats.com/dist/scala-with-cats.html) - Free e-book available in HTML, PDF, and ePUB formats introducing the Cats library.
- [Akka](https://akka.io/) and [Akka Streams](https://doc.akka.io/docs/akka/current/stream/index.html) - A toolkit for building highly concurrent, distributed, and resilient message-driven applications.
- [Alpakka](https://doc.akka.io/docs/alpakka/current/) - An Akka Streams-based library used to publish and subscribe to SQS, S3, Kafka, etc. with built-in back pressure support.
- [Slick](https://scala-slick.org/) - A reactive, functional object-relational mapper (ORM).
