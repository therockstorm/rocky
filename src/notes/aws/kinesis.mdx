- Data Firehose: move data from point A to B with optional transforms
- Data Analytics: analyze streaming data with, e.g., SQL, Java, etc.
- Data Streams: capture, process, and store data streams in real-time
- Video Streams
- Composed of producer, stream store, and consumer
- Streams have shards and retains data for 24 hours by default
- Data Firehose provides real-time data analysis with Lambda
- Analytics can be used with both to analyze data
